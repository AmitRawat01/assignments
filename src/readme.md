# Flipkart & Snapdeal Web Scraper 🛒

A Node.js-based web scraper using **Express, request, and cheerio** to extract product details from **Flipkart (Mobiles) and Snapdeal (T-Shirts)**.

---

## 🚀 Features
✅ Fetch product listings from Flipkart and Snapdeal  
✅ Extract details such as **title, price, rating, and product URL**  
✅ Fetch detailed Flipkart mobile product information (scrapes both **category** and **individual product pages**)  
✅ Asynchronous scraping for better performance
